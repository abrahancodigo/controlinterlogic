# Plan: Fix módulo Liquidación de Ruta no carga

## Problema
Al hacer clic en "Liquidación de Ruta" no pasa nada. El módulo no carga.

## Causa raíz
`const toCents` y `const fromCents` se declaran con `const` en **dos archivos**:
- `public/js/interlogic.js` líneas 10-11
- `public/js/liquidacion.js` líneas 6-7

Ambos se cargan en el mismo scope global. El segundo `const` lanza:
```
SyntaxError: Identifier 'toCents' has already been declared
```

Esto impide que **todo** `liquidacion.js` se ejecute, por lo que `window.Liquidacion` nunca existe.

## Fix
**Archivo:** `public/js/liquidacion.js`

Eliminar las 3 líneas duplicadas (líneas 6-8):
```js
// ELIMINAR:
const toCents = v => Math.round((parseFloat(v) || 0) * 100);
const fromCents = c => (c / 100).toFixed(2);

// Dejar solo:
const Liquidacion = {
```

`toCents` y `fromCents` ya están definidos en `interlogic.js` que se carga primero.

1. Abrir consola del navegador (F12) — no debe haber errores de SyntaxError
2. Hacer clic en "Liquidación de Ruta" — debe cargar el selector de rutas
3. Crear una ruta desde Control Interlogic y verificar que aparece en el dropdown
4. Seleccionar una ruta y verificar que carga el detalle de entregas
